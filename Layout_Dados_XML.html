Especificação de Integração JDE - Layout de Dados (CSV)

Público-alvo: Analistas de Negócios, Key Users e Equipe de Integração JDE.
Este documento especifica o layout comercial e técnico do arquivo de integração, gerado automaticamente para importação de notas fiscais no sistema JD Edwards.

1. Visão Geral

Devido à restrição nativa de caracteres na tabela do JDE, o arquivo final organiza os dados adotando um modelo linear e hierárquico. Toda Nota Fiscal é desmembrada em blocos de registros sequenciais. O primeiro campo de cada linha sempre indicará o tipo do bloco.

Padrões de Formatação:
  • Separador de Colunas: Ponto e vírgula (;)
  • Valores Monetários Vazios: Preenchidos com 0
  • Campos de Texto Vazios: Preenchidos com @
